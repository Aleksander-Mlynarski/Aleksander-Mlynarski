## About Me

I write engineering-grade C++ and Python for autonomous systems — currently building the navigation brain of an unmanned surface vessel at AGH Solar Boat, a student research team at AGH UST.

On that project I design and implement the full decision-making stack: behavior trees in BehaviorTree.CPP v4, sensor-driven waypoint logic, and ROS 2 node architecture that has to work on real hardware in competition conditions.

I also maintain a portfolio of projects in algorithms, numerical methods, computer vision, and C++ systems programming — skills that map directly onto perception, planning, and reliable onboard software.
